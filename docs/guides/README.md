## Getting Started

1. **Install [Freighter Wallet](https://www.freighter.app/)** - Browser extension for Stellar
2. **Connect to Astera** - Visit the app and connect your wallet
3. **Choose your role** - SME or Investor
4. **Follow the relevant guide above**
